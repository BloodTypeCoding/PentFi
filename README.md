# PentFi

Financial and accounting management software designed for Christian churches, which also features parishioner membership management. This software is delivered as a Web-based Service (SaaS). While it was primarily created for congregations of the United Pentecostal Church of Colombia (IPUC), it is available to all Christian denominations looking to track their accounting and finances in a systematized and organized manner.

## Technologies Used

- React
- TailwindCSS
- DaisyUI

## Setting-up

Clone the repository: 

```bash
git clone https://github.com/BloodTypeCoding/PentFi.git
```
Open the folder where the project is:

```bash
cd .\PentFi\
```
Install dependencies:

```bash
npm install
```
Run project

```bash
npm run dev
```
